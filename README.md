# proyecto-final-crossy-road
Este proyecto es un pequeño videojuego inspirado en Crossy Road, donde un axolote cruza calles y recolecta monedas. La idea principal fue crear un juego sencillo para practicar modelado 3D, animación, integración con Godot y funciones básicas de VR. El juego corre automáticamente y el jugador solo lo observa desde un visor VR.
